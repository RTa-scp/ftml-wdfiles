# Wdfiles Saving (FTML/Wikidot Static Previewer)
A static browsing site for files saved in FTML format, the SCP Foundation's markup language.

This repository uses pnpm for package management.

Install development dependencies and use
```bash
pnpm install
pnpm run dev
```
